# 🎉 Initial Release: FlowPKG (AeroPKG) v1.0

Welcome to the very first version of **FlowPKG** (AeroPKG)! 🚀
This is a modern, smooth Windows utility for analysing your PlayStation 4 `.pkg` files in the blink of an eye, without overwhelming your RAM.

## ✨ Key Features

* **Instant Analysis (Streaming)**: Files larger than 50 GB open immediately thanks to targeted binary reading.
* **Immersive "PS4-style" Interface**: Enjoy an elegant dark mode with smooth animations inspired by the console.
* **Advanced Metadata Extraction**: Retrieves the internal `param.sfo` (Title, Title ID, Required Firmware, Region, etc.) and displays the `icon0.png` icon.
* **FPKG Detection**: Automatically identifies whether your file is an official PKG or a Fake PKG.
* **Context Menu Integration**: Open your `.pkg` files directly from Windows Explorer with a right-click.

## 📦 Installation Instructions

1. Download the `FlowPKG.exe` file (or the `.zip` containing the executable) from the **Assets** below.
2. To enable the **right-click "Open with FlowPKG"** option in Windows:
   * Run `install_context_menu.reg` (make sure to open it in a text editor first to verify that the paths inside match the location where you placed the `.exe`!).

*(This release includes everything needed to use it as a standalone application on Windows.)*
